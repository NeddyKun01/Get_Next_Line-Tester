# Code of Conduct

## Our Pledge

We want this project to be welcoming, practical, and respectful for students,
reviewers, and maintainers.

Contributors are expected to communicate with patience and clarity, especially
when discussing failing tests or beginner mistakes.

## Expected Behavior

- Be respectful and constructive.
- Explain technical disagreements with examples.
- Keep feedback focused on the code, tests, and documentation.
- Assume good intent when someone is learning.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language.
- Personal attacks during code review.
- Publishing private information without permission.
- Repeated disruptive behavior after being asked to stop.

## Scope

This code of conduct applies to project spaces such as issues, pull requests,
documentation discussions, and related community conversations.

## Enforcement

Maintainers may remove inappropriate comments, close discussions, or restrict
participation when needed to keep the project healthy.
